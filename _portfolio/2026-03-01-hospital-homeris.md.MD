## 📝 Descripción del Proyecto
Este proyecto consistió en el diseño y despliegue de una arquitectura modular para el análisis de datos en el **Hospital Homeris**. El objetivo principal fue transformar datos complejos de salud en tableros accionables para la toma de decisiones estratégicas, permitiendo una visualización clara de indicadores críticos.

## 🧪 Ciencia de Datos Aplicada
Como líder técnico en analítica, implementé un ciclo de vida completo del dato:
* **Procesamiento Masivo:** Manejo y limpieza de datasets utilizando **Python (Pandas, NumPy)**
  
* **Modelado Predictivo:** Aplicación de técnicas de **Machine Learning** (Clustering) para identificar patrones y segmentar datos de pacientes
  
* **Visualización:** Despliegue de un dashboard interactivo en **Streamlit** para el monitoreo de indicadores clave en tiempo real
  
* **Arquitectura de Software:** Código estructurado de forma modular para facilitar el mantenimiento y la escalabilidad del análisis.

## 🚀 Impacto y Resultados
* **Automatización:** Implementación de una estructura de scripts que optimiza el flujo de trabajo desde la ingesta hasta el reporte
  
* **Decisiones Basadas en Datos:** Mejora en la interpretación de datos clínicos para el personal administrativo y médico al encontrar dos grupos de pacientes.
